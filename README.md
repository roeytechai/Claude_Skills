# Claude Skills

A collection of skills for Claude (Cowork, Claude Code, Claude.ai). Each skill lives in its own folder with full source, documentation, and an installable `.skill` package.

## Skills

| Skill | What it does | Install |
|---|---|---|
| [skill-validator](./skill-validator) | Audits and enhances your other Claude skills — evidence-based flaw reports, before/after demos, user-approved fixes only. Works on single skills, whole setups, and multi-agent teams. | [skill-validator.skill](./skill-validator/skill-validator.skill) |

*More skills coming.*

## Installing a skill

Download the `.skill` file from the skill's folder and open it in the Claude desktop app — click **Save skill**. For Claude Code, copy the skill's source folder into `~/.claude/skills/`.
